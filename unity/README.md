# Vector Flux Standalone (Unity Project)

This project is the standalone transition of the Vector Flux Skateboarding Game.

## Project Structure
- **Assets/Scripts**: Core gameplay C# scripts.
    - SkateController.cs: Physics-based movement (Rigidbody).
    - ProceduralSkater.cs: Stick figure movement and procedural rigs.
    - GrindRail.cs: Interaction logic for rails.
- **Assets/Prefabs**: (Empty) Ready for object creation.
- **Assets/Scenes**: (Empty) Ready for level design.

## How to Start
1. Open **Unity Hub**.
2. Click **Add** -> **Add project from disk**.
3. Select this unity/ folder.
4. Open the project and set the Render Pipeline to **URP** (Universal Render Pipeline) to maintain the high-contrast aesthetic.
